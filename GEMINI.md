# Project Overview

This project is a simple, static website for "Dungloe Hydraulics & Fabrication". It is built with plain HTML and CSS. The website displays the business's location, opening hours, and contact information. It is hosted on Netlify.

## Building and Running

### Running Locally

There is no build process for this project. To view the website locally, open the `public/index.html` file in a web browser.

### Deployment

This project is hosted on Netlify and is configured for continuous deployment. Pushing a commit to the `main` branch on GitHub will automatically trigger a new deployment.

## Development Conventions

The website is composed of static HTML and CSS files.

*   **HTML:** The main page is `public/index.html`. The 404 page is `public/404.html`.
*   **CSS:** Styles are located in `public/styles/main.css`.
*   **Images:** Images are stored in `public/images/`.

The styling is basic and follows a simple structure. There are no specific coding style conventions enforced.
