# Dungloe Hydraulics & Fabrication

This repository contains the source code for the company website for "Dungloe Hydraulics & Fabrication".

It is a simple, static website built with HTML and CSS.

## Content

The website displays:
- Business location via an embedded Google Map.
- Opening hours.
- Contact information (phone numbers and email).

## Development

To run the website locally, simply open the `public/index.html` file in a web browser.

## Deployment

This website is hosted on Netlify. Any commit pushed to the `main` branch will automatically trigger a new deployment.
